# Java Completo POO + Projetos

Este diretório contém uma coleção de programas desenvolvidos em Java como parte das práticas do curso "Java Completo POO + Projetos" da Udemy.

## Descrição

Os programas abrangem uma variedade de conceitos e práticas em programação orientada a objetos (POO) em Java. Cada programa aborda diferentes aspectos da linguagem e seu uso em projetos práticos.

## Conteúdo

- `Program01`: Calculadora de impostos ICMS e IPI.
- `Program02`: Manipulação de datas utilizando classes do pacote `java.time`.
- `Program03`: Cadastro e consulta de produtos com lista de entidades.
- `Program05`: Gerenciamento de produtos com adição e remoção de estoque.
- `Program06`: Cálculo da circunferência e volume de um círculo.
- `Program07`: Conversão de dólares para reais com taxa de IOF.
- `Program08`: Simulação de abertura de conta bancária com depósito e saque.
- `Program09`: Identificação de números pares em uma lista de números.
- `Program10`: Reserva de quartos em um Hotel.
- `Program11`: Demonstração de conceitos de Boxing e Unboxing.
- `Program12`: Demonstração de operações básicas com Listas em Java.
- `Program13`: Cadastro de funcionários e aumento de salário.
- `Program15`: Busca de um valor em uma matriz e identificação dos valores adjacentes.
- `Program16`: Simulação de posts e comentários em redes sociais.
- `Program17`: Sistema de pedidos de clientes com adição de itens.
- `Program18`: Comparação de áreas entre dois triângulos.

## Curso
